About Secure Messaging

Privacy-First Messaging

A zero-knowledge secure messaging service designed for maximum privacy and security. Messages are encrypted, ephemeral, and completely anonymous.

Features

  • End-to-end encrypted messages
  • One-time read and automatic deletion
  • No personal information required
  • No tracking or analytics
  • Transparent and secure
  • Rate limiting for security
  • Mobile and desktop friendly

Technology Stack

Spring Boot Java 17 PostgreSQL Thymeleaf Docker Maven

Transparency

This project prioritizes transparency in its security and privacy implementation. The architecture is designed with clear security boundaries and privacy-first principles.

Privacy & Security

Built with privacy as the foundation. No personal data is collected, stored, or tracked. Messages are encrypted at rest and in transit, and are automatically deleted after being read once. For more details, see our Privacy Policy.
